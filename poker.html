Нужна бесплатная Firebase чтобы все видели одни данные. Займёт ~3 мин.

Открыть Firebase Console →
«Добавить проект» → имя → отключи Analytics → создай
В меню: Сборка → Realtime Database → Создать базу данных → любой регион → «Начать в тестовом режиме»
Видна вверху страницы Realtime Database
Настройки проекта (⚙️) → «Общие» → добавь Web-приложение → скопируй apiKey и projectId
